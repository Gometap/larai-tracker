Under the hood

The "Invisible"
Interception Engine

Most trackers require you to use their SDK or wrap your client. Larai Tracker hooks directly into Laravel's ResponseReceived event, parsing token headers and body data in the background.

  • Works with OpenAI, Gemini, Claude, and more
  • Zero performance overhead (Asynchronous processing)
  • Secure, single-table database architecture 
// No changes to your code!
Http::withToken(config('openai.key'))
    ->post('https://api.openai.com/v1/chat/completions', [
        'model' => 'gpt-4o',
        'messages' => [['role' => 'user', 'content' => 'Hello!']]
    ]);

// Larai Tracker automatically captures:
# Model: gpt-4o
# Tokens: 42 Prompt / 128 Completion
# Cost: $0.00342 (Auto-calculated)
